# 璃奈板Project
“璃奈板”出自日本万代南梦宫集团旗下全媒体音乐偶像企划《LoveLive!虹咲学园学园偶像同好会》，是登场角色天王寺璃奈制作的舞台道具，可以在表演时随歌曲自动变换表情来向台下观众传达情感。本项目遵循AGPL-3.0协议开源，致力于面向LoveLive!系列企划粉丝打造一款能够完全还原动画中功能的、更加贴合用户实际使用需求的璃奈板产品。
